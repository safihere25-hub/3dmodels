# 3D Models Repository\n\nA collection of high-quality 3D models in GLB format, optimized for use with Lovable and other web-based 3D viewers.\n\n## 📁 Repository Structure\n\n```\n3dmodels/\n├── models/                 # GLB 3D model files\n│   └── overview-skeleton.glb\n├── docs/                   # Documentation\n│   └── model-catalog.md\n└── README.md\n```\n\n## 🎯 Models\n\n### Overview Skeleton\n- **File:** `models/overview-skeleton.glb`\n- **Description:** A detailed skeleton model suitable for anatomical or educational visualization\n- **Usage:** Can be directly imported into Lovable projects using Three.js or Babylon.js\n\n## 🚀 How to Use in Lovable\n\n### Method 1: Direct Raw URL\nReference the model using GitHub's raw content URL:\n\n```javascript\nconst modelUrl = \"https://raw.githubusercontent.com/safihere25-hub/3dmodels/main/models/overview-skeleton.glb\";\n```\n\n### Method 2: Using Three.js\n```javascript\nimport { GLTFLoader } from 'three/examples/jsm/loaders/GLTFLoader.js';\n\nconst loader = new GLTFLoader();\nloader.load('https://raw.githubusercontent.com/safihere25-hub/3dmodels/main/models/overview-skeleton.glb', (gltf) => {\n  scene.add(gltf.scene);\n});\n```\n\n### Method 3: Using Babylon.js\n```javascript\nBabylon.SceneLoader.ImportMesh(\"\", \n  \"https://raw.githubusercontent.com/safihere25-hub/3dmodels/main/models/\", \n  \"overview-skeleton.glb\", \n  scene\n);\n```\n\n## ✨ File Format\n\n- **Format:** GLB (GL Transmission Format Binary)\n- **Benefits:** \n  - Self-contained (includes geometry, textures, animations)\n  - Optimized for web delivery\n  - Fast loading times\n  - Widely supported across 3D engines\n\n## 📝 Notes\n\n- All GLB files are optimized for web usage\n- Models are stored in the `models/` directory\n- Use raw GitHub URLs for direct access from web applications\n- For large files, consider using Git LFS (Git Large File Storage)\n\n## 📄 License\n\nPlease specify the license for your models.\n\n## 🔗 Resources\n\n- [Three.js GLTFLoader Documentation](https://threejs.org/docs/#examples/en/loaders/GLTFLoader)\n- [Babylon.js SceneLoader Documentation](https://doc.babylonjs.com/features/featuresDeepDive/Babylon.js_and_WebXR/WebXR_LoaderDecorator)\n- [GLB File Format Specification](https://www.khronos.org/registry/glTF/specs/2.0/glTF-2.0.html)\n"
